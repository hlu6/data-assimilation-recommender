# Dynamic Recommender Product Showcase

This repository is set up to work well as a GitHub Pages portfolio project for a dynamic recommendation system.

## Positioning

This project presents a real-time recommendation concept that combines:

- temporal graph-based prediction
- online filtering updates after new interactions
- ranking-based evaluation for adaptive recommendation quality

The current site is written as a product showcase so it is easier to include in a resume, portfolio, or job application.

## Files

- `index.html`: one-page GitHub Pages site
- `styles.css`: visual design for the showcase
- `main.py`: existing placeholder Python file

## Publish On GitHub Pages

1. Create a GitHub repository for this project.
2. Upload these files to the repository root.
3. In GitHub, open `Settings > Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and the `/ (root)` folder.
6. Save the settings and wait for deployment.
